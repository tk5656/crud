# CRUD

## 概要

PHP と MySQL を使って、基本的な CRUD（作成・閲覧・更新・削除）処理を学習するために作ったアプリです。  
ログイン機能や画像アップロード、サムネイル生成など、Webアプリでよく出てくる処理を一通り実装しています。

---

## 機能

- ログイン（Cookie を利用）
- 一覧表示
- 新規登録
- 編集
- 削除
- 画像アップロード
- サムネイル自動生成（最大 150×100）
- 二重登録防止の工夫あり

---

## 使用技術

- PHP  
- MySQL  
- HTML  
- Sass  
- JavaScript（今後追加予定）  
- GD（画像処理）

---

## データベース

例：`items` テーブル

| カラム | 内容 |
|------|------|
